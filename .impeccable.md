## Design Context

### Users
- **Who**: 喜爱中式点心的消费者，以年轻到中年群体为主
- **Context**: 移动端为主场景，在手机上浏览和下单，碎片化购物
- **Job to be done**: 快速浏览精美的点心产品、便捷下单、追踪订单状态

### Brand Personality
- **3 words**: 温暖 · 传统 · 精致
- **Voice**: 亲切但不随意，像一位热情的点心师傅在介绍手艺
- **Emotional goals**: 看到就饿了的食欲感 + 家的温暖感

### Aesthetic Direction
- **Visual tone**: 暖色调为主（中国红、金色、暖白），干净现代的排版，参考喜茶/奈雪的国潮新中式风格
- **Photography**: 食物摄影是核心，暖光、近景、突出食材质感和蒸汽感
- **Typography**: 中文使用系统字体栈（-apple-system, "PingFang SC", "Microsoft YaHei"），西文使用 Geist Sans；标题可适度使用更有个性的字重
- **Spacing**: 宽松呼吸感，卡片式布局，圆角柔和（0.625rem base radius）
- **Theme**: Light mode 为主（暖白底），Dark mode 为辅
- **Anti-references**: 避免过于严肃的企业风格；避免信息过载的传统电商页面；避免冷色调

### Design Principles

1. **Food First** — 食物是主角。每个页面的视觉焦点应该是产品图片，UI 元素退让不抢戏。大图、高质量摄影、暖光氛围。

2. **Warm & Inviting** — 暖色调贯穿始终。用色彩、圆角、柔和阴影营造温暖感。避免尖锐边角和冷色调。间距宽松，给内容呼吸空间。

3. **Mobile-Native** — 移动端不是缩小版的桌面端。触控友好（最小 44px 触控区域）、单手可操作、滚动流畅。底部导航栏、大按钮、简洁表单。

4. **Progressive Disclosure** — 信息分层呈现，不一次性堆砌。首页突出精选，分类引导探索，详情页深入展示。减少认知负担。

5. **Delightful Details** — 适度的微交互提升体验。加入购物车的动效、下拉刷新、价格变化的过渡动画。但克制不花哨，服务于功能而非炫技。

### Component Guidelines
- **Cards**: 圆角、微阴影、hover 时轻微上浮。产品卡以图片为主（至少占 60% 面积）
- **Buttons**: Primary 用中国红填充色，Secondary 用暖金色边框。圆角柔和，点击区域充足
- **Navigation**: 移动端底部 Tab 导航（首页/分类/购物车/我的），桌面端顶部导航栏
- **Typography scale**: 标题 bold、正文 regular、辅助信息 muted-foreground small
- **Icons**: 使用 Lucide icons，线条风格，保持一致性
- **Images**: 产品图 1:1 或 4:3 比例，圆角裁切，懒加载 + blur placeholder
